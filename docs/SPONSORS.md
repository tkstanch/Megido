# Sponsor Megido

## Why Sponsor Megido?

Megido is an open security testing platform intended to help researchers and teams identify vulnerabilities responsibly. Sponsorship supports the sustainability of community-driven security tooling and increases practical impact for the broader research ecosystem.

## What Sponsorship Enables

Your support helps fund:

- More scanner modules and better verification quality
- Faster bug triage and fixes
- Better documentation and onboarding for new contributors
- Infrastructure and maintenance costs

## How to Sponsor

- GitHub Sponsors: <https://github.com/sponsors/tkstanch> (once enabled)
- Funding configuration reference: [../.github/FUNDING.yml](../.github/FUNDING.yml)
