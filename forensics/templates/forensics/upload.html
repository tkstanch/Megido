What happens after upload?

  • File is securely stored for analysis
  • SHA256 and MD5 hashes are calculated
  • File type and size are detected
  • First 256 bytes are extracted as hex sample
  • Device information is extracted (when available)
  • Results are displayed in a detailed report

Supported File Types

Disk Images:
  • Raw disk images (.dd, .img, .raw)
  • EnCase Evidence Files (.e01)
  • Advanced Forensic Format (.aff)
  • Virtual disk images (.vmdk, .vhd)
Other Formats:
  • Android/iOS backups (.ab, .ipa)
  • Log files (.log)
  • Memory dumps (.mem, .dmp)
  • Archive files (.zip, .tar, .gz)
